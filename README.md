# Task Management Application

A modern Task Management Application built using CodeIgniter 4 with authentication, AJAX CRUD operations, DataTables export functionality, and event-based task generation using hooks/events.

---

# Features

- User Registration & Login
- Authentication & Session Management
- User-specific Task Management
- Create, Read, Update, Delete Tasks
- AJAX-based CRUD Operations
- DataTables Integration
- Export Options:
  - Copy
  - CSV
  - Excel
  - PDF
  - Print
- Event/Hook-based Auto Task Creation on Login
- Bootstrap 5 Modern UI
- Responsive Design
- Validation & Error Handling

---

# Technologies Used

- PHP 8+
- CodeIgniter 4
- MySQL
- Bootstrap 5
- jQuery
- AJAX
- DataTables
